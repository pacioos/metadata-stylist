XSLT at the Pacific Islands Ocean Observing System (PacIOOS)

This directory contains Extensible Stylesheet Language Transformations (XSLT)
for transforming XML documents into human-readable formats like HTML and
plain text. These are used to provide alternative views of our FGDC and ISO
metadata XML documents and to translate to-and-from FGDC and ISO metadata
standards.

METADATA VIEWS:

For HTML and plain text views of our metadata in both ISO and FGDC formats
using these XSLT stylesheets, you can visit a website whose location is based
on each dataset's identifier:

    http://pacioos.org/metadata/[dataset identifier].html

For example:

    http://pacioos.org/metadata/NS01agg.html
    http://pacioos.org/metadata/NS01agg.txt
    http://pacioos.org/metadata/NS01agg.xml

    http://pacioos.org/metadata/NS01agg.html?format=fgdc
    http://pacioos.org/metadata/NS01agg.txt?format=fgdc
    http://pacioos.org/metadata/NS01agg.xml?format=fgdc
